# Bash Database Management System

A simple Database Management System built using **Bash scripting**.
This project simulates basic database operations such as creating databases, connecting to them, and inserting records into tables using command-line tools.

## 📌 Features

* Create and manage databases
* Connect to a specific database
* Create tables
* Insert records into tables
* Store table data in files
* Simple command-line interface

## 🛠 Technologies Used

* Bash Scripting
* Linux File System
* Command Line Interface (CLI)

## 📂 Project Structure

```
.
├── main.sh
├── connect.sh
└── insert_func.sh
```

### File Description

| File           | Description                                          |
| -------------- | ---------------------------------------------------- |
| main.sh        | Main script that starts the database system          |
| connect.sh     | Handles connecting to a selected database            |
| insert_func.sh | Contains functions for inserting records into tables |

## ▶️ How to Run

1. Clone the repository:

```
git clone <repo-link>
```

2. Navigate to the project folder:

```
cd bash-database
```

3. Give execution permission:

```
chmod +x *.sh
```

4. Run the main script:

```
./main.sh
```

## 💡 Example Operations

* Create a new database
* Connect to a database
* Create a table
* Insert records

All data is stored as files in the Linux file system.
